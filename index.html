Project Overview

Financial technology continues to change how people pay, save, invest, and manage money. At the same time, stronger cybersecurity has become essential because financial systems are frequent targets for phishing, identity theft, fraud, and ransomware. This site presents a basic overview of these topics in a simple website format for IT 3203.

This is a class project for IT 3203.
